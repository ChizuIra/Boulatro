La boule du joueur fait la course avec la boule de l'ordinateur.
Si la boule de l'ordinateur arrive a 10 Tours avant le joueur, ce dernier perd.
Si la boule du joueur arrive a  10 Tours en premier, le joueur a alors jusqu'a la fin de la course de l'ordinateur pour faire un maximum de Tours.
> Chaque tours supplementaire donne +1$

A chaque fin de Course le joueur fait face a une boutique, dans laquelle il peut utiliser ses $ pour acheter des bonus.

Interet a la balatro + 


Début de course : QTE de lancer de bille ( en mode flippeur )
Le joueur a le droit à un nombre de replay, pour soit re essayer de gagner, soit essayer de faire un meilleur score.
Le dernier résultat est conservé


Boucle de Gameplay : 
-> Lancement de la boule (Avec QTE Flippeur)
-> le joueur observe les boules faire la course
-> Defaite -> relance
-> victoire-> Choisis entre accepter sa victoire ou relancer pour essayer de faire mieux

-> Defaite -> plus de relance -> fin de partie
-> Victoire acceptée ou plus de relance -> Calcul du gain d'argent puis intérêt

-> Shop : Le joueur dépense son argent pour améliorer sa boule via des jokers et autres bonus
-> Round Suivant. ( retourner en haut )
